## Features

- **AI Assistant** — Generate and explain SQL queries with built-in AI. Describe what you need in plain English.
- **Schema Diagrams** — Visualize database relationships with beautiful, auto-generated ER diagrams.
- **Inline Editing** — Edit data directly in the results table. Changes are validated and saved instantly.
- **Query History** — Never lose a query again. Full history with search, favorites, and quick recall.
- **Multiple Databases** — Connect to SQLite and PostgreSQL. Switch between workspaces effortlessly.
- **Native Performance** — Built with Tauri and Rust. Starts in under a second, uses minimal memory.

## Why Keisen?

- No Electron. No bloat. Just a native app that feels right.
- Write SQL with confidence — autocomplete, syntax highlighting, and AI assistance.
- Manage multiple database connections in one place.
- Works offline. Your data stays on your machine.
