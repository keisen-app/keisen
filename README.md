# Keisen

A fast, lightweight SQL client for macOS, Windows, and Linux.

Built with Tauri, React, and Rust.

## Features

- **AI Assistant** — Generate and explain SQL queries with built-in AI. Just describe what you need in plain English.
- **Schema Diagrams** — Visualize your database relationships with auto-generated ER diagrams.
- **Inline Editing** — Edit data directly in the results table. Changes are validated and saved instantly.
- **Query History** — Full history with search, favorites, and quick recall.
- **Multiple Databases** — Connect to SQLite and PostgreSQL. Switch between workspaces effortlessly.
- **Native Performance** — Built with Tauri and Rust. Fast, lightweight, and native to your system.
